Расчеты значений частот cдля HUS

1. Для издания ЦАПом HUS звука требуемой частоты t_freq (target frequency) сэмплом, оцифрованным
с частотой d_freq (digitized frequency) при частоте смешения HUS mix_greq (mixing frequency) необходимо
в регистр "дробного шага" записать значение fract_step, расчитанное по формуле:

fract_step = d_freq/mix_freq*t_freq/523.251

Частота оцифровки сэмплов в ProTracker = 16726Гц.
Частота смешения HUS (по умолчанию)    = 48000Гц.

При таких условиях, выбранных для мод-плеера, формула вычисления дробного шага следующая:

fract_step = t_freq/1501,6171230419705847184024871458

2. Музыкальный модуль содержит в паттернах не номера нот и не частоты, которые нужно проиграть, а значение периода period, 
которое нужно записать в регистр звукового чипа Paula, чтобы получить частоту, соответствующую ноте.
Значение выходной частоты по периоду (для ProTracker) расчитывается по формуле:
t_freq = 7159090.5/32/period.
Соответственно, расчет значения дробного шага по периоду следующий:
fract_step = d_freq/mix_freq*7159090.5/32/period/523.251.
Для вышеуказанных параметров:
fract_step = 148,98709843677436832418858253496/period
